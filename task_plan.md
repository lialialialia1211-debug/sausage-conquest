# 烤香腸征服戰 — 體驗優化改進計畫

## 目標
串連已有系統，讓玩家每個烤制決策產生漣漪效應，提升核心循環的爽感與策略深度。

---

## Phase 1: 核心循環強化（烤制體驗）
> 優先做投入產出比最好的改動，不新增內容，純機制優化

### Task 1.1: Perfect 連擊系統
- **檔案**: `src/scenes/GrillScene.ts`
- **內容**: 追蹤連續 perfect 出餐次數，觸發 combo 加成
  - 連續 3 根 perfect → 1.2x 收入 combo
  - 連續 5 根 perfect → 「神之手」全場客人耐心回滿 + 1.5x 收入
  - combo 斷裂時顯示回饋（「combo 中斷！」）
- **狀態**: [x] 完成 (2026-04-16)

### Task 1.2: 出餐瞬間視聽強化
- **檔案**: `src/scenes/GrillScene.ts`, `src/objects/SausageSprite.ts`
- **內容**:
  - Perfect 出餐：金色閃光 + 畫面微震 + 放大動畫 (0.3s)
  - Combo 觸發：連擊數字飛出 + 螢幕邊緣金色光暈
  - Carbonized：黑煙粒子 + 攤位微暗
  - 客人反應文字氣泡（perfect → 驚嘆、burnt → 捂鼻）
- **狀態**: [x] 完成 (2026-04-16)

### Task 1.3: 動態營業時長（tier-based）
- **檔案**: `src/scenes/GrillScene.ts`, 常數區
- **內容**: GAME_DURATION 改為依 tier 計算
  - tier 1-3: 75 秒（新手期，壓力小）
  - tier 4-6: 90 秒（中期，標準節奏）
  - tier 7-9: 105 秒（高級區，更多客人更多機會但也更累）
- **狀態**: [x] 完成 (2026-04-16)
